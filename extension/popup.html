Request Monitor

Version 1.0

This extension monitors web requests to specific domains and sends them to a backend system for analysis and alerting.

How it works:

  • Automatically fetches monitoring configuration from your backend
  • Monitors network requests to configured domains
  • Sends request data to backend for pattern analysis
  • Runs silently in the background
Privacy: Only monitors domains specified by your backend configuration. All data is sent to your configured backend server only.
